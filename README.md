# Facebook Graph API Project

## Overview
This project is a web application that interacts with the **Facebook Graph API** to fetch data from Facebook pages, posts, and user interactions. Built using **HTML, CSS, JavaScript (Node.js/Express)**, it demonstrates API requests, responses, and data handling on the frontend.

---

## Features
- **Login User Accounts
- **Display Username and Profile
- **Logout
- **Light/Dark Mode

---

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/facebook-graph-api.git

```
2.Navigate to the project folder:

cd facebook-graph-api

3. Install dependencies (Node.js required):

npm install

4. Create a .env file in the root directory with your Facebook credentials:

ACCESS_TOKEN=your_facebook_access_token
PORT=3000

5. Start the backend server:

node server.js

6. Open index.html in your browser or serve via a local server.
